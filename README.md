# Car_Price_Predictor
A linear regression problem based on car dekho dataset.


## About Dataset
This dataset contains information about used cars.

The columns in the given dataset are as follows:
1. name
2. year
3. selling_price
4. km_driven
5. fuel
6. seller_type
7. transmission
8. Owner
9. mileage
10. engine
11. max_power
12. torque
13. seats
